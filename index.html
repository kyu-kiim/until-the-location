The signal arrives. some are discarded. some are averaged. and a reference point is created. In between, the position is not determined. but gradually converges.

This work reveals the state before this convergence.

Click and select. connected lines. and repeated patterns are traces before they become positions. and each path is a residue of data that remains unreduced.

The position here is not a result, but a Process.
